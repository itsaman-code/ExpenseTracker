# 💰 Expense Tracker (Java Console Application)

A simple yet powerful **Java-based Expense Tracker CLI application** that helps users manage daily expenses efficiently using categories, search, filter, sorting, and analytics features.

---

## 🚀 Features

- ➕ Add expenses with category and name
- 📋 View all expenses in a formatted table
- 🔍 Search expenses by name, category, or amount
- 🎯 Filter expenses (above, below, between range)
- 🔃 Sort expenses:
  - Highest to lowest
  - Lowest to highest
  - Name A-Z / Z-A
  - Category-wise
- 🗑 Delete expenses by ID
- 💰 View total spending summary
- 🧠 Input validation for safer user experience
